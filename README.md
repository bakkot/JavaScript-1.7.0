# JavaScript-1.7.0, built for wasm

This is JavaScript as it was in the Firefox 2 days, forked from [this repo](https://github.com/Historic-Spidermonkey-Source-Code/JavaScript-1.7.0) for continuity reasons but you can get the source [straight from Mozilla](https://ftp.mozilla.org/pub/js/).

I had Claude get it building on my Mac (only very minor tweaks required!) and for WebAssembly (somewhat more tweaks required). See [./js/src/build-wasm](./js/src/build-wasm) for its docs (which I have not read) or point your browser at [the online demo](https://bakkot.github.io/JavaScript-1.7.0/).
